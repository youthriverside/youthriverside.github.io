    我们经常会有一些问题

    我的网站为啥这么慢

    许多站长

    首先，如果你的钱很够，你完全可以搞一台高配服务器。
    有句话说得好 需求+钱=解决
    我们需要做的，
    就是在有限的资源内，
    达到尽量快的速度。
    有一些大佬的网站，可以说是秒开。
    我们应该怎么优化呢？

    WP优化有什么妙用？不知道就丢脸了
    你知道WP优化吗？WP优化最近为什么这么火？
    众所周知，WP优化是最近非常火的一个词。
    那么WP优化到底是什么梗呢？
    相信大家对WP优化都很熟悉，
    WP优化就是我们每天都会经常遇到的，但是WP优化是怎么回事呢？
    其实之前并没有太多人知道WP优化，
    大家可能会感到很惊讶，WP优化为什么是这样的？
    WP优化究竟为什么火起来了呢？
    但事实就是这样，小编也感到非常惊讶。
    以下就是小编为大家带来的的关于WP优化是什么意思，WP优化是什么梗的内容。

    营销号专用开头

    这两个插件都带有压缩功能
    也可以把这些代码放到_function.php_里

    add_action('init', 'wpjam_gzip_compression');function wpjam_gzip_compression(){    if (substr_count($_SERVER['HTTP_ACCEPT_ENCODING'], 'gzip')) {        ob_start('ob_gzhandler');     }else{        ob_start();     }}htacce

    再简单一点,
    直接使用插件Gzippy
    我还是比较推荐直接用缓存插件自带的。

    缓存插件

    缓存插件有很多，
    比如说自称很好的LiteSpeed Cache
    甚至说比WP rocket好
    目前比较流行的插件有
    WP Fastest Cache
    WP Super Cache
    WP Rocket
    WP Total Cache
    里面的设置有以下小建议：
    CSS合并 开启后可能会出现加载时间过长
    然后所有东西一下子出来的现象
    所以你觉得一点一点出现好
    还是一起出来好
    举个栗子
    你访问个站点，一直等什么也看不到。
    另一个站点一点点加载出来。
    你会对哪个充满希望？
    JS合并 开不开无所谓
    合并后可以有效减少请求。
    所有选项都自己尝试一遍
    有一点需要注意
    缓存插件不是越多越好
    但是越多越慢是可能。
    有一些插件彼此兼容 比如Autoptimize和WP rocket
    有时候你会遇到加缓存没有裸奔快的情形
    这时候可以去掉插件

    CDN

    我们可以使用CDN（内容分发网络）来加速站点
    比如说我现在就在用腾讯云的CDN
    开通后每个月50G持续6个月
    到期后每月10G
    国内多地延迟在10ms左右

    还有一些公司如Cloudflare、百度云加速、栅格云都可以
    以下是缓存规则
    /wp-admin 是一定要回源的
    所有文件 几天都行
    CSS和图片类的 几天都行
    .gz .xml .php直接回源

    还有就是看清是国内还是国外流量
    这一点很重要

    其他

    推荐使用Memcached和OPcache扩展
    修订版本如果没有需要可以删除
    但是如果有人复制你文章可能很难拿出证据。
    国内尽量删除谷歌字体
    一些插件有这个功能
    插件不要太多
    主题要使用正规来源的
    不会什么要折腾
    不折腾出来一个博客咋来成就感
